# Robotics Roadmap

A comprehensive 16-week robotics learning roadmap and tracker. This repository contains a single-page interactive HTML application (`roadmap.html`) that allows you to track your progress through various phases of learning robotics.

## Overview
The 16-week roadmap is divided into several intensive learning phases:
- **Phase 1 — Foundation**: Linux, Git, C++ Basics, C++ OOP, and Advanced C++.
- **Phase 2 — Robotics Core**: Python for Robotics, ROS2 (Nodes, Topics, Sub/Pub), and Robot Math (Kinematics).
- **Phase 3 — Control + Simulation**: PID Control, Gazebo, URDF, and Quadruped Simulation.
- **Phase 4 — Hardware + Ship**: Embedded Systems (Arduino/Teensy), Building a 2DOF Robot Arm, and Resume/Application prep.

## Features
- **Progress Tracking**: Click on tasks to mark them as completed. The tracker automatically calculates your percentage of completion.
- **Local Storage**: Your progress is saved locally in your browser, so you can close the page safely without losing data.
- **Offline Capable**: Includes a service worker making this a fully functional Progressive Web App (PWA) that loads offline.

## Usage
Simply download or clone this repository and open `roadmap.html` in your web browser. 
No build steps or dependencies are required.
